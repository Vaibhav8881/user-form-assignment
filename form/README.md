# backend-interview-assignment

## Using GIT

### clone
- Command
```
git clone git@github.com:Vaibhav8881/user-form-assignment.git
cd interview-assignments
```

### Using Maven 

### build
- Command
```
mvn clean install -DskipTests
```

### run
- Command
```
java -jar target/form-0.0.1-SNAPSHOT.jar
```
